My First Week Learning HTML!

Posted on January 15, 2024

Wow, what an exciting week! I've just started my programming journey and I'm learning HTML. It's amazing how you can create web pages with just text and tags.

Something really exciting happened today - I created my first proper HTML page with headings, paragraphs, and even a list! Here's some of what I learned:

<h1>This is a heading</h1>
<p>This is a paragraph</p>
<ul>
  <li>List item 1</li>
  <li>List item 2</li>
</ul>

The most important thing I learned is that HTML gives structure to web pages. It's like building the skeleton of a website before you make it look pretty with CSS.

I'm looking forward to next week when I start learning about links and images. My goal is to build a personal website by the end of the month!

CSS Magic - Making Things Look Pretty

Posted on January 22, 2024

This week has been all about CSS (Cascading Style Sheets), and I have to say - it's like magic! HTML creates the structure, but CSS makes everything look beautiful.

I learned how to:

  • Change font styles and sizes
  • Add colors to text and backgrounds
  • Adjust line height for better readability
  • Use classes and IDs to style specific elements

Here's some CSS I wrote to style this blog:

body {
  font-family: "Georgia", serif;
  line-height: 1.6;
  background-color: #f8f9fa;
}

h1 {
  color: #2c3e50;
  text-align: center;
}

The coolest part was learning about the box model - how every element has margin, border, and padding. Я поняла где и как распологаются все части у сайтов
